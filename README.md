# Genius scripts

Scripts for Genius transcriptions

Both require exporting a spreadsheet (Spreadsheet 2) of all your transcriptions using Genius Artwork Extractor

<img width="500" height="204" alt="image" src="https://github.com/user-attachments/assets/8e2b7324-258f-4225-ae4c-db55b02eece6" />

## genius_not_awarded

Shows all unawarded transcriptions, excluding instrumentals. Useful if you want an additional IQ boost for unawarded scribes

<img width="1113" height="626" alt="image" src="https://github.com/user-attachments/assets/7093f0f4-f849-4ff2-9adf-263cf981f74c" />

This will also save in a .txt file in case you need to come back to it.

## genius_pageview_list

Exports all your awarded transcriptions in a nice, formatted list similar to [this page](https://genius.com/Para-accomplishments-and-transcriptions-annotated). Minimum views is 5000, but can be adjusted.

<img width="1113" height="626" alt="image" src="https://github.com/user-attachments/assets/702e8d0f-ad9a-469f-b74a-0d8d786d5e60" />

This will also save in a .txt file for easy copy and pasting onto a new page
